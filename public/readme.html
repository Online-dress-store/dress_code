What are you selling?

Additional pages and how to use

  • Wishlist: like items and add to cart.
  • Sell: list your own items.
  • Account: personal details, purchases.
  • Admin: activity log + manage products.
  • Quick View: modal with product details and recommended accessories with external buy links.

What was hard?

Authentication flow with JWT cookies and session management, ensuring all API routes return JSON-only responses (no HTML), implementing real-time cart/wishlist synchronization across pages using custom events and localStorage, building comprehensive admin activity logging with prefix filtering, and creating robust data persistence system with file-based storage for users, products, and activity logs.
