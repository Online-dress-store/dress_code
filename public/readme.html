What are you selling?

Pre‑loved and new dresses (Evening, Summer, Holiday, Casual).

Additional pages and how to use

  • Wishlist: like items and add to cart.
  • Sell: list your own items.
  • Quick View: modal with product details and recommended accessories with external buy links.
  • Style Quiz: login-gated quiz at /quiz. Submit preferences → fetch personalized /api/quiz/recommendations and render cards.

What was hard?

Creating the Virtual Try-On page was the most challenging part of the frontend development. I attempted to build a feature where users could upload their photo and overlay a dress image with drag/scale controls, but struggled with complex image processing requirements like background removal, EXIF orientation handling, and canvas manipulation. The technical complexity of real-time image editing, file upload handling, and creating a smooth user experience for image positioning proved too difficult to implement successfully within the project timeline. This taught me the importance of properly scoping feature complexity and the value of focusing on core functionality first.
